# HTTP Client App
The purpose of this project is to play with the HttpClient class.

## Tasks:
- [x] Make an HTTP call (read a website)
- [x] Serialization 
- [x] Deserialization
- [x] Make an API call (read an API endpoint)
- [x] Deserialize response
